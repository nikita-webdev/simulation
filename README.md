# Project Simulation

Console-based implementation of a simulation in 2D space.

---

| Entity    | Icon |
|-----------|------|
| Herbivore | 🐑   |
| Predator  | 🐆   |
| Grass     | 🌾   |
| Rock      | 🏔️  |
| Tree      | 🌳   |
| Empty     | ⬛    |

---

## Features

- 🐑 Herbivores roam and graze on grass.
- 🐆 Predators hunt herbivores.
- 🌾 Grass grows and spreads over time.
- 🏔️ Rocks and 🌳 Trees form natural obstacles.
